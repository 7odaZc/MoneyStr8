# MoneyStr8 — Personal Finance Tracker (UI Prototype)

A Vite + React UI prototype for a personal finance tracker (Phase 2 deliverable).

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite.

## Notes

- Static prototype: no backend, no persistence.
- Demo navigation:
  - `/` landing
  - `/login`, `/register`
  - `/dashboard`, `/transactions`, `/settings`
